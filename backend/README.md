# dwengo-1-backend

## Project setup

```shell
npm install
```

Setup the environment variables in a `.env` file in the root of the project. You can use the `.env.example` file as a template.

### Development

```shell
npm run dev
```

### Production

```shell
npm run build
npm run start
```

### Tests
Voer volgend commando uit om de unit tests uit te voeren:
```
npm run test:unit
```
